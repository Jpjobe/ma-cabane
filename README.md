# Ma Cabane 🏠❄️

Jeu de survie isométrique inspiré de Whiteout Survival.

## Jouer

👉 [Jouer en ligne](https://TON-USERNAME.github.io/ma-cabane/)

## Description

Survie dans un environnement hivernal hostile :
- Collecte de ressources (bois, pierre, blé, poisson)
- Construction d'abris, moulins, boulangeries, poissonneries
- Gestion de la faim et du froid
- 4 saisons avec changements visuels
- Loups en hiver
- Habitants secondaires

## Installation locale

Ouvre simplement `index.html` dans ton navigateur, ou utilise un serveur local.

## Technologies

- HTML5 Canvas
- CSS3
- JavaScript Vanilla
- PWA (installable sur mobile)
